
# MV* 框架

MV*模式指的其实就是对代码进行按照功能划分模块的方式！

MVC:
Model: 和数据相关的代码，负责提供数据的！
View: 和展示相关的代码，负责展示数据的！
Controller: 负责将数据转交给视图，同样负责从视图中收集数据，转交给Model

MVP:
Model: 和数据相关的代码，负责提供数据的！
View: 和展示相关的代码，负责展示数据的！
Controller: 负责将数据转交给视图，同样负责从视图中收集数据，转交给Model


MVVM:
Model:
View:
View-Model: 这个部分是由框架实现的，不需要程序员再进行开发了，就可以自动实现数据到视图，以及视图到数据的流程！
