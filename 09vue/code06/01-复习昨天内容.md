### 组件之间的通信

组件如果需要相互传递数据，就需要用组件通讯技术

什么样的组件关系是父子组件关系？


Vue.component("组件A", {
  template: "<组件B />"
  components: {
    "组件B": {

    }
  }
})


1. 父组件传子组件
   属性传值：

2. 子组件传父组件
   事件传值：
      1. 先在父组件中声明一个函数，这个函数用来接收数据的，所以需要一个形参
      2. 通过@将这个函数绑定给子组件标签
      3. 在子组件中，调用this.$emit("绑定的函数名", 要传递的数据)
